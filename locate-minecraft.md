## **Windows 10/11**:

<ol>
  <li> Press Windows + R.
  <li> Type in %AppData%.
  <li> If there is an option, select Roaming. If not, skip this step.
  <li> Locate the .minecraft folder.
</ol> 

## Mac OS

<ol>
  <li> Open Finder.
  <li> Press Command + Shift + G.
  <li> Type in ~/Library.
  <li> Navigate to Application Support.
  <li> Locate the .minecraft folder.
</ol>
